## 📖 About

A high-performance external enhancement utility for GTA V single-player mode. Provides real-time memory interaction through a modern ImGui-based overlay with customizable themes.

Built with performance in mind — minimal FPS impact, no injection required.

---

## ✨ Features

- **God Mode** — Complete player invincibility
- **Infinite Ammo** — Never reload again
- **Vehicle Spawner** — 300+ vehicles with preview
- **Teleport** — Instant travel to any waypoint on the map
- **Weather Control** — Rain, thunder, snow, clear sky at will
- **Wanted Level Editor** — Clear or max out your wanted stars instantly
- **Time Manipulation** — Freeze time, speed up, or slow down
- **Customizable UI** — Multiple themes, adjustable transparency
